# 🖥️ Hacker System Monitor (Web Edition)

A cool browser-based system monitor with Matrix-style animations and hacker aesthetic.

## ✨ Features
- 🌧️ **Matrix Rain Background** – Live raining characters
- 💻 **System Info** – OS, hostname, architecture, time
- 🔴 **Hacker Sequence** – Fake IP scanning animation
- 📊 **Progress Bar** – Monitoring animation
- 💬 **Rotating Quotes** – Hacker-themed quotes every 10 seconds
- 🎮 **Konami Code Easter Egg** – Type ↑↑↓↓←→←→ ba
- 📱 **Responsive** – Works on mobile too!

## 🚀 Quick Start
1. Download `index.html`
2. Double-click to open in your browser
3. Enjoy! No server needed

## 🎮 Easter Egg
Type the Konami code: **↑ ↑ ↓ ↓ ← → ← → b a**

## 🤓 Why I Built This
Because I wanted to feel like a hacker without actually hacking anything.
